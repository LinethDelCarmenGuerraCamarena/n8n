# 🦷 Asistente de Citas Dentales con n8n

El sistema permite agendar, consultar y gestionar citas de manera automatizada, integrando servicios de mensajería y lógica de negocio sin necesidad de intervención manual. A través de workflows, se procesan las solicitudes de los usuarios, se validan horarios disponibles y se generan respuestas dinámicas.

⚙️ Características principales
- Automatización del agendamiento de citas
- Validación de disponibilidad de horarios
- Respuestas automáticas a usuarios
- Integración con APIs y/o bases de datos
- Flujo visual y escalable con n8n
